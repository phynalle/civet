# civet(cv)
civet is a command line tool which has compatibility with cat. it colorize the contents of file along to the extension of each file.

## Usage
cv [-n] [file.. ]

## Lastest Version
###  Supported Langauges
* Rust
* Go
* C/C++
* Python
* Haskell
* Javascript
* TOML
* JSON

### Theme
* Monokai (default)
* Kimbie Dark
* Solarized Dark
